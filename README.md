# Media-Manager

Media-Manager is a web-based application for managing video streams with MediaMTX and FFmpeg.

## Features

-   **Backend:** Node.js, Express.js, MediaMTX, FFmpeg
-   **Frontend:** Bootstrap, Vanilla JavaScript
-   **Responsive Design:** Light and dark mode
-   **Resource Monitoring:** CPU, RAM, GPU usage
-   **Task Management:** Add, modify, start, and stop video tasks in real-time
-   **Video Previews:** HSL previews of active streams
-   **Configuration:** MediaMTX and application settings
-   **Dockerized:** Containerized with Docker for easy deployment
