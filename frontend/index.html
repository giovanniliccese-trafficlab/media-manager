Media-Manager v1.0.0

Professional media streaming management platform

Features
  • Multi-task video streaming management
  • RTSP and HLS protocol support
  • Playlist creation with multiple videos
  • Scheduled streaming with cron jobs
  • Real-time system monitoring (CPU, RAM, GPU)
  • Live HLS preview in web interface
  • File browser for USB drives and SMB shares
  • Backup and restore configuration
  • Responsive dark/light theme interface
  • Error logging and monitoring
Technology Stack
Backend
  • Node.js 24
  • Express.js 4.18
  • FFmpeg 7.x
  • MediaMTX Latest
  • Docker & Alpine Linux
Frontend
  • Bootstrap 5.3.3
  • Vanilla JavaScript ES6+
  • Chart.js 4.4
  • HLS.js
  • Bootstrap Icons
Streaming Protocols
  • RTSP: Real-Time Streaming Protocol (Port 554)
  • HLS: HTTP Live Streaming (Port 8888)
  • WebRTC: Web Real-Time Communication (Port 8889)
